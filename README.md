# DotVader

## Introduction

 Ce repo contient mes "dotfiles" - les fichiers de configurations utilises par
 plusieurs cli tools(git, vi, zsh)

## Usage

Je vous recommende de regarder les fichiers et de copy paste ce que vous avez vraiment besoins
dans _vos_ dotfiles.
